属性指令(Attribute directive): 用于改变组件的外观或行为

ngStyle指令： 用于设定给定 DOM 元素的 style 属性

绑定常量

[ngStyle]="{{content1}}"

绑定表达式

[ngStyle]="{{content2}}"

[ngStyle]="{{content3}}"

[ngStyle]="{{content4}}"

[style.property语法]

绑定常量

[style.color]="'red'"

绑定表达式

[style.color]="fontColor"

[style.color]="getColor(fontColor)"

[style.color]="fontColor === 'blue' ? 'blue' : 'red'"

ngClass指令：用于动态的设定 DOM 元素的 CSS class

绑定常量

[ngClass]="{{text3}}"

[ngClass]="{{text4}}"
